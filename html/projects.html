Open Source Projects

virtdc                                                          

CLI(Command Line Interface) to statically place and schedule and dynamically scale and migrate Vitual Machines (VMs) to optimize resource utilization by using efficient VM algorithms while meeting the SLAs of users in cloud. Python package is available for this CLI

You can find the detailed project information here

vmonere                                                                   

A cloud monitoring CLI which provides per second monitoring data of the domain(Guest) to the monitor server. vmonere agent installed in the guest reports the usage to the vmonere listener through sockets and log the usage in text file and json file. Graphical interface is developed in java swings and html to plot the graph for the usage.

You can find the vmonere demo monitor here

gitnotus                                                                   

A github event management CLI. It sends notification to the contributors of the repositories with file information, for example newly added file names, deleted file names, modified file names etc. This CLI uses git web hooks and a web server to listen to the hook URL
